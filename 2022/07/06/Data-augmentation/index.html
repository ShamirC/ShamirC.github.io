数据增强

数据是机器学习，特别是深度学习这类数据驱动方法达到高精度的最重要因素之一。

在图像分类，语义分割等图像处理方面，数据增强（Data augmentation)是指对图像进行翻转，添加高斯噪声等。数据增强的作用之一是可以提高模型的泛化性。

数据增强可以分为

  • 线上增强，在数据输入模型之前，对数据进行翻转，添加噪声等，这种增强方法不会改变数据量。
  • 线下增强，对数据进行翻转，添加噪声等操作后，保存到本地，这种增强方法会增加数据量。后续与原始数据一起作为输入训练网络

色彩变换

为了适应实际场景中，拍摄角度，光照等对图像色彩的影响。常用的还有图像色调（hue)，亮度（brightness)，饱和度（saturation)，对比度（contrast) 的随机变换等。

RGB、HSV、HSL 颜色模型

RGB

RGB 是我们接触最多的颜色空间，由三个通道表示一幅图像，分别为红色(R)，绿色(G)和蓝色(B)。这三种颜色的不同组合可以形成几乎所有的其他颜色。RGB 颜色空间利用三个颜色分量的线性组合来表示颜色，任何颜色都与这三个分量有关，而且这三个分量是高度相关的，所以连续变换颜色时并不直观，想对图像的颜色进行调整需要更改这三个分量才行。

HSV

在图像处理中使用较多的是 HSV 颜色空间，它比 RGB 更接近人们对彩色的感知经验。

HSV 表达彩色图像的方式由三个部分组成：

  • Hue（色调、色相）
  • Saturation（饱和度、色彩纯净度）
  • Value（明度）

Hue: 用角度度量，取值范围未为0~360°，颜色圆环上所有的颜色都是光谱上的颜色，hue=0 表示红色，hue=120 表示绿色，hue=240 表示蓝色等。

hsv 圆柱体的竖切剖面（ Hue=60 )

S (saturation)表示接近光谱色的程度，值越高颜色越深，饱和度越低越接近白色

V（Value) 表示明度，决定颜色空间中颜色的明暗程度，值越高表示越亮，0表示纯黑色

HLS

HLS 与 HSV 类似， 不同的是，HSL 最后一个分量是 亮度 (light)， HSV 是明度（value）

  • Hue, 色调，
  • Light, 亮度，亮度为100，表示白色，亮度为0，表示黑色
  • Saturation，饱和度，

RGB --> HSV

\[ R^{'} = R/255\\ G^{'} = G/255\\ B^{'} = B/255\\ \]

\[ C_{max} = max(R'G'B')\\ C_{min} = min(R'G'B')\\ Delta = C_{max} - C_{min} \] 计算 Hue (色调) \[ \\ H = \begin{cases} a^{\circ}\quad \Delta = 0\\ 60^{\circ}\times \left(\frac{G{}'-B{}'}{\Delta} + 0 \right)\quad ,C_{max}=R{}'\\ 60^{\circ}\times \left(\frac{B{}'-R{}'}{\Delta} + 2 \right)\quad ,C_{max}=G{}'\\ 60^{\circ}\times \left(\frac{R{}'-G{}'}{\Delta} + 4 \right)\quad ,C_{max}=B{}'\\ \end{cases} \\ \] 计算 Saturation（饱和度） \[ \\ S = \begin{cases} 0\quad C_{max}=0\\ \frac{\Delta}{C_{max}}\quad C_{max}\ne0 \end{cases} \\ \] 计算 Value (明度) \[ V = C_{max} \]

HSV --> RGB

\[ C = S\\ H' = H/60\\ X = C(1-|H'mod2-1)\\ \\ (R,G,B) = (V-C)(1,1,1) + \begin{cases} (C,X,0)\quad (0 <= H' < 1)\\ (X,C,0)\quad (1 <= H' < 2)\\ (0,C,X)\quad (2 <= H' < 3)\\ (0,X,C)\quad (3 <= H' < 4)\\ (X,0,C)\quad (4 <= H' < 5)\\ (C,0,X)\quad (5 <= H' < 6)\\ \end{cases} \]
