Metric criteria

机器学习中评价模型精度的指标有

  • Precision, Recall, F1
  • overall accuracy
  • ROC-Curve, PR-Curve
  • Average Precision (AP), mAP
  • Matthew correlation coefficient (MCC)
  • OIS, ODS

Confusion Matrix

Precision, Recall, F1, Ovarll accuracy 等都可以基于该矩阵(表格) 进行计算

公式 含义
Precsion P = TP / (TP + FP) 模型预测中，预测为正例中实际也为正例的所占的比例
Recall R = TP / (TP + FN) 样本中，正样本被正确预测所占的比例
F1 F1= 2*P R / (R + P) or TP/(2\TP+FN+FP) Precision 和 Recall 的调和平均值
IoU（Intersection over Union) / Jaccard IoU = TP / (TP + FP + FN) 语义分割模型的重要衡量指标之一，是预测值和真实值的交集和并集之比
Matthew correlation coefficient (MCC)(马修斯相关系数) 二分类模型结果的评估指标之一，值在[-1，+1], 由于Precision,recall只考虑了正类，而忽略了负类的表现，为了解决这个问题，可以将prediction与真实结果看为两个0-1分布，然后用马修斯相关系数衡量两个分布的相似性。当FP=FN=0 即完全预测正确时 MCC=1，当完全预测错误时MCC=-1，此时将标签逆转即可。
Dice-coefficient 2*\ X∩Y\ /(\ X\ + \ Y\ ) 用于度量两个样本的相似性，dice 等同于 F1。也可以作为loss

除了评价模型对测试样本的指标之外，如

  • 执行时间
  • 内存占用

这些衡量模型执行速度，内存占用的指标在某些情况下也需要考虑。

常用的精度评价曲线有 ROC (Receiver Operating Characteristic, 受试者曲线）, P-R 曲线

针对线性目标提取的指标

公式 含义
OIS (Optimal Image Scale) 针对数据集的一张图片，遍历[0,1]的阈值，使该图片F1值最大，将数据集中所有图片的最佳F1值求平均值即得到 OIS
ODS (Optimal Dataset Scale) 遍历[0-1] 所有的阈值作为整个数据集分类的阈值，若某一个阈值下，使得整个数据集F1值最大时，该F1值即为 ODS。

P-R 曲线

选取不同的阈值对预测样本进行划分，可以计算出不同的 precision 和 recall, P-R曲线就是以不同阈值下取得的 precision(纵坐标）, recall（横坐标） 绘制曲线。平衡点（BEP）是P=R时的取值，如果这个值较大，则说明学习器的性能较好。。

ROC曲线

ROC 曲线最早应用于雷达信号检测区域，用于区分信号与噪声。很多分类器是为测试样本产生一个实值或概率预测，然后根据一个阈值对样本进行分类。若想得到更好的 precision, 则可以提高阈值，尽可能得使得对每一个预测值都有充分的自信。在不同阈值下得到的 FPR, TPR. 然后以 FPR 为横坐标，TPR 为纵坐标绘制 ROC 曲线。TPR, FPR 都是从实际样本出发，因此不受样本不平衡的影响

AUC（曲线下面积，the area under the ROC curve) 是指 ROC曲线下与横坐标轴构成的区域面积， AUC表示了可分离的程度或测度。ROC 曲线纵坐标轴是 TPR，TPR 越高表示模型正确预测正例的概率越大，因此理想情况就是，曲线离 (0,1) 点越近，AUC面积越接近 1，模型性能越好。若连接对角线，AUC面积正好是 0.5 对角线的实际含义是：随机判断响应与不响应，正负样本覆盖率应该都是 50%，表示随机效果

公式 含义
Average Precision (AP) 等于P-R曲线下面积，一般通过逼近的方法求曲线下面积吗
mAP
Ture Positive Rate (TPR) (正阳率，灵敏度) / Sensitivity TPR = TP / (TP+FN) 与 recall 相同，
Speficity(特异度) S = TN / (FP + TN)
False Positive Rate (TNR) (假阳率，特异度) / Specificity FPR = 1- S = FP / (FP + TN) 从公式可以看出，TPR, FPR是分别从实际的正样本和负样本中来观察相关概率问题的，因此不受样本不平衡的影响
